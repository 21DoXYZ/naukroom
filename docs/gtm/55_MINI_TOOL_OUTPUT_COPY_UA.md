# 55_MINI_TOOL_OUTPUT_COPY_UA.md

**Артефакт:** Структура і copy результату mini-tool  
**Мета:** Визначити що показує result screen і яким має бути tone  
**Пріоритет:** Must have  
**KPI:** result_view > 30s (engagement), CTA click rate після result

---

## Структура результату (AuditLiteResult)

```typescript
interface AuditLiteResult {
  problems: string[]      // рівно 3 проблеми
  strength: string        // 1 сильна сторона
  rewrittenBio: string    // 1 переписаний Bio
  offerDirection: string  // 1 напрямок оферу
  suggestedCTA: string    // 1 рекомендований CTA для Reels
}
```

## Tone of voice результату

- Конкретний, не загальний
- Прямий, але не агресивний
- Відчуття "це про мене", не шаблон
- Без жаргону і buzzwords
- Без "AI-мова" (не "генерую", "обробляю", "алгоритм")
